Git基本操作

2021.07.03

克隆及提交

  1. clone到本地 git clone address；address为需要clone的地址
git clone https://github.com/FDXN/FDXN.github.io/
  1. git status命令可以查看文件状态(可不执行，只是用于查看文件状态)
git status
  1. git add ,提交所有变化 ( -A 命令区分大小写)
git add .
  1. git commit -a -m"内容" 提交到本地仓库
git commit -m'2.5.0代码更新'
  1. git push 上传更新到git版本库
git push

创建新分支

原来分支叫 v2.5.0，现在需要创建一个新的 v2.5.1分支 操作如下

  1. 切换至需要克隆的分支v2.5.0
git checkout v2.5.0
  1. 拉取该分支最新的代码
git pull
  1. 基于v2.5.0创建新分支v2.5.1
git checkout -b v2.5.0
  1. 将本地分支v2.5.1push到远程
git push origin v2.5.1
